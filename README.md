# nuimo-node-demo
Sample apps to demonstrate functionality of Nuimo BTLE device

##Setup
run `npm install` 

##Demo with Logging:
run `node index.js`

##Demo of volume control (OSX):
run `node volume.js`

##Demo of led matrix:
run `node ledmatrixdemo.js`
